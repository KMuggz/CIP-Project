# 🐍 Snake Sprint — Pygame Edition

A modern twist on the classic Snake game, built with Python and Pygame as my Project for **Stanford Code in Place 2025**.

---

## 🎮 Features

- Snake must collect meals and return to base
- Countdown timer per delivery
- Obstacles and game-over conditions
- High-value meals (2x / 3x points)
- Adjustable difficulty (Easy / Medium / Hard)
- Music support and restart controls

---

The music used is Copyright free from (https://freemusicarchive.org/genre/Soul-RB/)
Song : Dreams in slow motion by Snoozy Beats (lo-fi instrumental)

---

## 🎹 Controls

| Key | Action |

| ←  ↑  ↓  → | Move the snake |
| 1 / 2 / 3 | Select difficulty |
| R | Restart the game |
| M | Toggle background music |
